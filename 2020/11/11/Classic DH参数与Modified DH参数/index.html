Classic DH参数与Modified DH参数

本文最后更新于：2020年11月11日 下午

前言

DH参数算是机器臂中最最基础的内容了，但是在学习的过程中发现不同教材建立的规则不尽相同，在这里对比一下Classic DH参数和Modified DH参数，二者的区别知乎上有挺多介绍了，这里主要说明一下二者各自的建立过程，尤其是参数确定的部分。

根据连杆坐标系和关节对应关系的不同，DH 建模法可以分为传统 DH (Classic DH) 和改进 DH (Modified DH)，简单来说，Classic DH的ZiZ_iZi​是位于关节i+1i+1i+1上的，而Modified DH的ZiZ_iZi​是位于关节iii上的

相比来说，Modified DH更符合人们的认知习惯，且比Classic DH更加通用，具体可参见这个问题下的答案，《Introduction to Robotics Mechanics and Control》用的就是Modified DH，但依然有很多的教材采用的啥Classic DH，所以同时掌握两种方法我认为还是有必要的。

Classic DH

前提

相当于人为的引进了两个约束条件，使得原来需要六个参数才能描述的位置和姿态，仅仅用四个参数就可以描述；

具体地，是指XiX_iXi​必须与Zi−1Z_{i-1}Zi−1​垂直且相交。

建立各坐标系

  1. 中间连杆
    1. 从base到ending对关节进行编号，顺序为1~n；
    2. 建立ZiZ_iZi​~Zn−1Z_{n-1}Zn−1​：其中ZiZ_iZi​轴与关节i+1i+1i+1的运动轴（旋转或滑动）对齐，方向任选（方向的选择会影响参数的正负）
    3. 建立各个坐标系的原点O1O_1O1​~On−1O_{n-1}On−1​ ：其中第i个坐标系的原点在Zi−1Z_{i-1}Zi−1​与ZiZ_iZi​的交点，或Zi−1Z_{i-1}Zi−1​&ZiZ_iZi​公垂线与Zi−1Z_{i-1}Zi−1​的交点；
    4. 建立X1X_1X1​~Xn−1X_{n-1}Xn−1​：其中XiX_iXi​与Zi−1×ZiZ_{i-1}\times Z_iZi−1​×Zi​向量同向，且背离关节i−1i-1i−1；当Zi−1Z_{i-1}Zi−1​与ZiZ_iZi​平行时，位于他们的垂直平面，且背离关节i−1i-1i−1；
    5. 建立Y1Y_1Y1​~Yn−1Y_{n-1}Yn−1​：根据右手螺旋定则确定；
    6. 至此，坐标系{1}~{n-1}建立完成。
  2. 首部连杆
    1. 建立Z0Z_0Z0​，其中Z0Z_0Z0​轴与关节111的运动轴（旋转或滑动）对齐，方向任选（方向的选择会影响参数的正负）；
    2. 为了使问题简化，X0X_0X0​和Y0Y_0Y0​的选择保证初始位置时关节111的关节变量（joint variable）为0.
  3. 末端连接器（the end-effector）
    1. 固连于手部的坐标系由末端执行器的几何形状确定，通常如下规则：
    2. 建立ZnZ_nZn​：沿着Zn−1Z_{n-1}Zn−1​，并指向远离机器人的方向；
    3. 建立坐标系原点OnO_nOn​：根据末端执行器的几何形状确定；
    4. 建立XnX_nXn​：垂直于Zn−1Z_{n-1}Zn−1​和ZnZ_{n}Zn​；
    5. 建立YnY_nYn​：根据右手螺旋定则确定；
    6. 至此，坐标系{n}建立完成。

建立各参数

  1. Joint angle θi\theta_iθi​：绕着ZiZ_iZi​轴，从Xi−1X_{i-1}Xi−1​到XiX_iXi​的角度，当关节iii是转动轴时，它是关节变量（joint variable）；
  2. Joint distance did_idi​：沿着Zi−1Z_{i-1}Zi−1​轴，从Xi−1X_{i-1}Xi−1​到XiX_iXi​的距离。当关节iii是平移轴时，它是关节变量（joint variable）；
  3. Link length aia_iai​：沿着XiX_iXi​轴，从Zi−1Z_{i-1}Zi−1​到ZiZ_iZi​的距离；
  4. Link twist angle αi\alpha_iαi​：绕着XiX_iXi​轴，从Zi−1Z_{i-1}Zi−1​到ZiZ_iZi​的角度。

坐标系{i-1}到坐标系{i}的变换

  1. 绕Zi−1Z_{i-1}Zi−1​旋转θi\theta_iθi​角度，使Xi−1X_{i-1}Xi−1​与XiX_iXi​同向；
  2. 沿Zi−1Z_{i-1}Zi−1​平移did_idi​距离，使Xi−1X_{i-1}Xi−1​与XiX_iXi​在同一直线上；
  3. 沿XiX_iXi​平移aia_iai​距离，使坐标系{i-1}原点与坐标系{i}原点重合；
  4. 绕XiX_iXi​旋转αi\alpha_iαi​角度，使Zi−1Z_{i-1}Zi−1​与ZiZ_iZi​重合，至此两个坐标系重合；

前提

相当于人为的引进了两个约束条件，使得原来需要六个参数才能描述的位置和姿态，仅仅用四个参数就可以描述；

建立各坐标系

  1. 中间连杆
    2. 从base到ending对关节进行编号，顺序为1~n；
    2. 建立Z1Z_1Z1​~Zn−1Z_{n-1}Zn−1​：其中ZiZ_iZi​轴与关节i的运动轴（旋转或滑动）对齐，方向任选（方向的选择会影响参数的正负）

    4. 建立X1X_1X1​~Xn−1X_{n-1}Xn−1​：其中XiX_iXi​与Zi×Zi+1Z_{i}\times Z_{i+1}Zi​×Zi+1​向量同向，且背离关节iii；当ZiZ_{i}Zi​与Zi+1Z_{i+1}Zi+1​平行时，位于他们的垂直平面，且背离关节i；
